# Programming

## Compilators
```
# Go
$ go run hello.go

# C
$ gcc hello.c -o helloc

# C++
$ g++ hello.cpp  -o hellocpp

# Rust
$ rustc pizza.rs
$ cargo run
```

## Hello World!
### Go-lang
```go
package main
import 'fmt'

func main(){
  fmt.Print("HELLO")
  fmt.Printf("HELLO")
}
```
### C
```c
#import <stdio.h>

int main(){
  printf("HOLA");
  return 0;
}
```
### C++
```c++
#import <iostream>
using namespace std;

int main(){
  cout << HI << endl;
  printf("HI");
  return 0;
}
```
### Rust-lang
```rust
fn main(){
  println!("pizza")
}
```
